# Desafio de Projeto - Criar Uma Página Web Utilizando as Tags Aprendidas

## Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio de projeto da Formacão HTML da DIO para criar uma página web utilizando as principais tags HTML. O objetivo foi reforçar os conceitos de HTML, aplicando as tags aprendidas em uma estrutura de página organizada e funcional.

A página foi dividida em diferentes seções, abordando temas relacionados à história dos computadores, clientes e servidores, a estrutura básica de um HTML e a diferença entre listas ordenadas e não ordenadas. Cada seção contém informações concisas sobre os tópicos, explicadas de forma simples para facilitar a compreensão de quem estiver lendo.

## Estrutura do Projeto

A estrutura da página HTML é simples, mas aborda diversos conceitos importantes:

- **História dos Computadores**: Uma breve explicação sobre a evolução dos computadores, desde os primeiros dispositivos até as tecnologias modernas.
- **O que são Clients?**: Definição e funcionamento dos clientes, com foco na comunicação entre dispositivos e servidores.
- **O que são Servers?**: Explicação sobre o papel dos servidores na arquitetura client-server e a forma como eles fornecem recursos para os clientes.
- **Como é a estrutura básica de um HTML?**: Apresentação da estrutura fundamental de uma página HTML, explicando os elementos `DOCTYPE`, `html`, `head` e `body`.
- **Entendendo a diferença entre listas ordenadas e não ordenadas**: Demonstração das diferenças entre as tags `<ol>` (listas ordenadas) e `<ul>` (listas não ordenadas), e como elas são utilizadas no HTML.

## Objetivo

O principal objetivo deste projeto foi aplicar os conceitos aprendidos sobre HTML, criando uma página simples, porém informativa, que explica de forma clara e objetiva temas essenciais para quem está começando a aprender sobre desenvolvimento web. Além disso, o projeto serve como base para a compreensão de como as tags HTML são usadas na prática.

## Tecnologias Usadas

- **HTML**: Linguagem de marcação usada para estruturar o conteúdo da página.

## Conclusão

O desafio foi uma excelente oportunidade para reforçar e aplicar os conhecimentos adquiridos em HTML, proporcionando uma base sólida para os próximos passos no desenvolvimento web. A criação de uma página simples, mas rica em conteúdo, demonstrou como as tags HTML podem ser utilizadas para organizar e exibir informações de maneira eficaz.
